### Bundler

Add this line to your application's Gemfile:

```ruby
gem "ruby-magicwrite"
```

And then execute:

$ bundle install

### Gem install

Or install with:

$ gem install ruby-magicwrite

and require with:

```ruby
require "magicwrite"
```
